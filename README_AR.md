# نظام نور الذهبي لإدارة الموارد (Golden Noura ERP)

## نظرة عامة
نظام متكامل لإدارة تأجير العمالة يتضمن:
- إدارة العمال والعملاء والعقود
- نظام الرواتب والمصروفات
- نظام الفواتير والمحاسبة
- نظام الصلاحيات والأدوار
- تقارير مالية متقدمة

## المتطلبات
- Python 3.8 أو أحدث
- Node.js 18 أو أحدث
- npm أو yarn

## التثبيت والتشغيل

### 1. تشغيل الخادم الخلفي (Backend)

افتح نافذة أوامر وتوجه إلى مجلد backend:

```bash
cd golden_noura_erp/backend
```

ثم قم بتثبيت المكتبات المطلوبة:

```bash
pip install -r requirements.txt
```

بعد ذلك، قم بتشغيل الخادم:

```bash
uvicorn main:app --reload --port 8000
```

### 2. تشغيل الواجهة الأمامية (Frontend)

افتح نافذة أوامر جديدة وتوجه إلى مجلد frontend:

```bash
cd golden_noura_erp/frontend
```

ثم قم بتثبيت المكتبات المطلوبة:

```bash
npm install
```

بعد ذلك، قم بتشغيل خادم التطوير:

```bash
npm run dev
```

### 3. التشغيل السريع

يمكنك تشغيل كلا الخادمين دفعة واحدة باستخدام ملف start.bat:

```bash
cd golden_noura_erp
start.bat
```

## تسجيل الدخول

### الافتراضي
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

### الوضع دون اتصال (Offline Mode)
إذا كان الخادم غير متاح، يمكنك استخدام:
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

## الميزات الرئيسية

### 1. نظام الترحيب
- عرض اسم المستخدم الكامل عند تسجيل الدخول
- عرض صورة المستخدم الشخصية
- رسالة ترحيب باللغة العربية

### 2. إدارة المستخدمين
- إنشاء مستخدمين جدد
- تعيين أدوار وصلاحيات
- رفع صورة المستخدم

### 3. لوحة القيادة
- إحصائيات شاملة
- رسوم بيانية تفاعلية
- تقارير فورية

## حل المشاكل الشائعة

### مشكلة الاتصال بالخادم
1. تأكد من أن الخادم الخلفي يعمل على المنفذ 8000
2. تحقق من ملف `.env` في مجلد frontend:
   ```
   VITE_API_URL=http://localhost:8000
   VITE_API_TIMEOUT=10000
   VITE_OFFLINE_LOGIN=true
   ```

### مشكلة في عرض الصور
1. تأكد من وجود مجلد `static/avatars` في backend
2. تحقق من صلاحيات المجلد

### مشكلة في اللغة
- النظام يدعم اللغة العربية بشكل افتراضي
- يمكنك تغيير اللغة من زر الكرة الأرضية في صفحة تسجيل الدخول
